    Brod-Roy 9/1/2022 MarkDown

    # 1、标题

    MD的标题一共有六级标题，语法基本一致。

    语法：

     # 一级标题
    1

    示例：

    # 示例标题

    快捷键：Ctrl + 1 (1-6)

    # 2、列表

    # 无序列表

    语法：

    + 列表内容
    1

    + - * 等符号都可以生成无序列表

    示例：

    列表内容

    快捷键：Ctrl + shift + ]

    # 有序列表

    语法：

    1. 列表内容
    1

    示例：

    1.列表内容

    快捷键：Ctrl + U

    # 嵌套列表

    语法：

    1. 一级列表内容

    + 二级列表内容

    + 二级列表内容
    1
    2
    3
    4
    5

    一级列表后回车，点击删除键，根据语法即可生成二级列表

    示例：

    1. 一级列表内容
      • 二级列表
      • 二级列表

    快捷键：

    # 3、引用

    语法：

    > 引用内容

>> 引用内容
    1
    2
    3

    示例：

    引用内容

    引用内容

    快捷键：Ctrl + shift + Q

    # 4、代码块

    # 少量代码

    语法：

    `代码内容`
    1

    示例：

    代码内容

    快捷键：无

    # 大量代码

    语法：

    ```代码内容```
    1

    示例：

    public static void main(String[] args) {
        System.out.println("hello world");
}
    1
    2
    3

    快捷键：Ctrl + shift + K

    # 5、链接

    此处一般无需自己书写，复制进来即可，他会帮你截取

    语法：[此处填超链接的文字描述](此处填超链接地址)

    示例：

    介绍 — Vue.js (vuejs.org) (opens new window)

    快捷键：Ctrl + K

    # 6、下划线

    下划线

    快捷键：Ctrl + U
